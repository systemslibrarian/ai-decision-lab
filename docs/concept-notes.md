# Concept Notes

AI Decision Lab illustrates the connection between two major ideas.

## 1. Discrete Search
Used for:
- graph traversal
- navigation
- shortest path planning
- obstacle avoidance

Examples:
- DFS
- BFS
- A*

## 2. Continuous Optimization
Used for:
- learning
- control
- trajectory refinement
- minimizing cost functions

Examples:
- gradient descent
- path smoothing
- turn minimization

## Robotics connection

Robots often:
1. find a feasible route
2. improve it
3. execute it under motion constraints

This project makes that pipeline visible.
