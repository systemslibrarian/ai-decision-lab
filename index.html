Find A Number In A List

This section shows two ways a computer can look for one number in a list. One checks every item. The other sorts the list first and cuts the search area in half each step.

Target: — Comparisons: 0 Ready —

What These Buttons Mean

🔎 Check One By One

The computer starts at the first item and keeps checking until it finds the match.

✂️ Cut Search In Half

After sorting the list, the computer keeps splitting the remaining choices in half to find the answer faster.

🔍 Deep Search

This tries one route for a long time before backing up and trying another.

🌊 Shortest Steps

This spreads out evenly from the start and is good for finding the path with the fewest steps.

💰 Cheapest Route

This pays attention to expensive areas and tries to find the route that costs the least overall.

🏃 Fast Guess

This rushes toward the goal based on what looks close, even if that guess turns out to be wrong.

⭐ Best Route

This balances distance so far with a smart guess about what looks promising next.

📉 Find Low Spot

This keeps moving downhill to find a low point, like a ball rolling into a valley.

🌡️ Escape Trap

This sometimes allows a worse move now so it can avoid getting stuck in a bad place later.

🧬 Evolve Answers

This tries many possible answers, keeps the stronger ones, and mixes them to make better ones.
